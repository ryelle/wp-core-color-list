WordPress Core color groupings

A list of all colors in WordPress CSS, matched against a set of "brand colors." The distance between each color is measured, and the CSS color is matched to the closest brand color. This uses Lab to calculate color difference. There is a distance cutoff, some colors are far away from all provied brand colors. These are listed at the end of the page, under "outliers."

These colors were extracted from WordPress using the css-audit tool. 

yarn run css-audit --colors wordpress-trunk/src/(wp-admin|wp-includes)/**/*.css --format=json
